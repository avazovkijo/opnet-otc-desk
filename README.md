# OTC Desk

High-density OPNet trading terminal for OTC quote lanes, spread comparison, and settlement flow.

## Live

- App: https://opnet-otc-desk.vercel.app

## Stack

- React
- TypeScript
- Vite
- WalletConnect for OPNet wallet shell

## Notes

- Current version is a premium frontend shell with demo/live contract source handling.
- Built as one of the three flagship apps in the OpenNet portfolio.

## Local

```bash
npm install
npm run dev
```
